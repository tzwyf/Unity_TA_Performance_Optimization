Unity 粒子系统与透明层优化（尽量不改变效果）

【核心原则】
减少“同一像素被重复绘制”的次数
优化重点不是 Triangle，而是 FillRate / Overdraw

==================================================
一、减少粒子数量（低损优化）
==================================================

1. 降低 Spawn Rate
- Rate over Time: 100 → 40
- Burst 数量减少
- 优先减少“持续生成”

2. 缩短生命周期
- Start Lifetime: 2s → 1.2s
- 减少历史粒子堆积

3. 使用 Flipbook 替代大量粒子
- Texture Sheet Animation
- 一个动画粒子替代多个静态粒子

4. 远距离自动减少数量
- 距离越远 Spawn 越少
- 小粒子远处直接 Cull

--------------------------------------------------

优化目标：
“少而精”
不要“多而糊”

==================================================
二、降低粒子透明区域（收益最大）
==================================================

1. 裁剪贴图透明边缘
错误：
512x512 中间只有一小团内容

正确：
裁掉外围透明区域

效果：
FillRate 可下降 30%~60%

2. 减小粒子屏幕覆盖面积
错误：
10个超大烟雾

正确：
30个较小烟雾

3. 使用 Tight Mesh / Tight Packing
减少无效透明像素

4. 避免超大Quad
大面积透明Quad是 Overdraw 杀手

--------------------------------------------------

优化核心：
减少“透明空白像素”

==================================================
三、减少粒子叠加（Overdraw核心）
==================================================

1. 减少透明层数
目标：
同一区域控制在 2~4 层

避免：
10层以上透明叠加

2. 缩短粒子停留时间
旧粒子尽快消失

3. 拉开粒子空间分布
避免大量粒子堆在同一区域

4. 控制爆炸类特效
不要：
一瞬间大量大粒子重叠

--------------------------------------------------

优化核心：
减少“同一像素重复绘制”

==================================================
四、控制场景透明层数
==================================================

1. 能不用 Transparent 就别用
优先：
Opaque
Alpha Clip
Dither

避免：
Alpha Blend

2. 草/树叶/栅栏
使用：
Alpha Clip

不要：
Alpha Blend

3. 大面积透明物体谨慎使用
包括：
烟雾
云
UI
玻璃
水面

4. 避免透明物体互相覆盖
透明层越多
GPU越贵

--------------------------------------------------

透明性能排序：

Opaque
↓
Alpha Clip
↓
Alpha Blend
↓
复杂PBR透明

==================================================
五、Shader优化（非常关键）
==================================================

避免：

- PBR Particle
- Refraction
- Distortion
- GrabPass
- 多层Noise
- 复杂Lighting

推荐：

- Unlit Particle
- Simple Lit
- Mobile Shader

--------------------------------------------------

原因：

Overdraw时：
每一层都会重复执行Shader

Shader越复杂
成本越高

==================================================
六、Soft Particle优化
==================================================

Soft Particle 很贵

原因：
需要读取Depth Texture

建议：

- 只给关键特效使用
- 大面积烟雾慎用
- 移动端尽量关闭

==================================================
七、LOD与距离优化
==================================================

近景：
高质量粒子

中景：
减少数量
减少透明

远景：
Flipbook
Billboard
直接Cull

--------------------------------------------------

工业级方案：

近处好看
远处便宜

==================================================
八、GPU排查流程（实战）
==================================================

Step1：
Scene View → Overdraw

Step2：
找到最红区域

Step3：
关闭单个粒子系统

Step4：
观察 GPU ms 变化

Step5：
优先优化：

- 大面积
- 长生命周期
- 多层叠加
- 高透明覆盖

==================================================
九、最终优化优先级（推荐顺序）
==================================================

【第一梯队（收益最大）】

1. 裁透明区域
2. 减少屏幕覆盖面积
3. 减少透明层数

【第二梯队】

4. 减少生命周期
5. 降低Spawn Rate
6. Flipbook替代

【第三梯队】

7. Shader简化
8. 减少SoftParticle
9. LOD/Cull

==================================================
十、一句话总结
==================================================

真正高性能的粒子系统核心不是：

“粒子少”

而是：

“同一个像素不要被反复绘制”
